# Träningsplan: Vecka 2 (26 dec - 31 dec)

**Mål för veckan:**
*   **Löpning:** Öka långpassets distans och introducera ett tröskelpass.
*   **Styrka:** Fokusera på pull-up progression och bygga råstyrka.

---

### **Fredag 26/12: Styrka (Överkropp & Pull-up Fokus)**
*   **Bänkpress:** 5 set x 5 reps @ **65 kg**.
*   **Latsdrag:** 4 set x 8 reps @ **45-50 kg** (öka om det känns lätt).
*   **Negativa Pull-ups:** 3 set x 3-5 reps (hoppa upp och håll emot på vägen ner, 3-5 sekunder).
*   **Skivstångsrodd:** 4 set x 8 reps @ **55 kg**.
*   *Syfte:* Bygga överkroppsstyrka och specifik styrka för pull-ups.

### **Lördag 27/12: Kondition (Lätt distans)**
*   **Pass:** 30 minuter lugn jogg.
*   *Syfte:* Spola ur benen och underhålla basen.

### **Söndag 28/12: Kondition (Långpass)**
*   **Pass:** **10 km** på löpband. Håll samma lugna, jämna tempo som förra långpasset.
*   *Syfte:* Progressiv ökning av uthållighet.

### **Måndag 29/12: Styrka (Underkropp & Plyo)**
*   **Marklyft:** 5 set x 5 reps @ **107.5 kg**.
*   **Lateral Jumps:** 3 set x 8 reps/sida.
*   **Plyo Push-ups (Explosiva armhävningar):** 3 set x 5-8 reps (tryck ifrån så händerna lämnar golvet).
*   **Good Mornings:** 3 set x 10 reps @ **40 kg**.
*   **Plankan:** 3 set x max tid.
*   *Syfte:* Bygga råstyrka och stabilitet.

### **Tisdag 30/12: Kondition (Tröskelpass)**
*   **Pass:** 10 min uppvärmning. **20 minuter** löpning i ett "bekvämt ansträngande" tempo (ca 6:00-6:15 min/km). 10 min nedvarvning.
*   *Syfte:* Förbättra din förmåga att hålla ett högre tempo över längre tid.

### **Onsdag 31/12 (Nyårsafton): Vila**
*   **Fokus:** Vila och förbered för det nya året.
